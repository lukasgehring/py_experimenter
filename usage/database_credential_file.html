Database Credential FileÔÉÅ

When working with MySQL as a database provider, an additional database credential file is needed, containing the credentials for accessing the database:

[CREDENTIALS]
host = <host>
user = <user>
password = <password>

By default, this file is located at config/database_credentials.cfg. If this is not the case, the corresponding path has to be explicitly given when executing PyExperimenter.
