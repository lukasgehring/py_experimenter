process_logs(logs: Dict[str, Dict[str, str]]) → None

Appends logs to the logtables. Raises InvalidLogFieldError if the given logs are invalid. The logs are of the following structure: Dictionary keys are the logtable_names (without the prefix table_name__). Each key refers to a inner dictionary with the keys as columnsnames and values as results.

Parameters:

logs (Dict[str, Dict[str, str]]) – Logs to be appended to the logtables.
